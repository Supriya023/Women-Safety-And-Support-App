#Adhira – Women Safety & Support App

Adhira is a modern safety application built to provide immediate assistance and support in emergency situations. The platform connects users with nearby volunteers and trusted contacts through real-time SOS alerts and location sharing.

🔹 Key Features:

* 🚨 SOS Emergency Alert with live location
* 📍 Real-time location tracking
* 🤝 Volunteer assistance system
* 💬 Chatbot support for guidance
* 🧑‍🤝‍🧑 Community interaction
* 📊 User & volunteer dashboards

🔹 Tech Stack:

* Frontend: Flutter
* Backend: Node.js, Express
* Database: MongoDB

Adhira aims to create a safer environment by enabling quick communication, support, and response during emergencies.
